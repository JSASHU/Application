# Demo App #

### What is this repository for? ###

* Demo App
* 1.0.0

### How do I get set up? ###

* clone the repo
* `cd` into the folder
* run `npm install`

### Start Mysql DB
* run `mkdir data && mongod --dbpath=data`  
  In a terminal window and keep it running

### To start the application
* run `gulp develop`

### Contribution guidelines ###

* Run `npm run pretest`
* Code review
